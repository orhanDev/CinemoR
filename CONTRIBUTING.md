# Contributing to CinemoR

Thank you for your interest in contributing to CinemoR!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/CinemoR.git`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Commit: `git commit -m "Add your feature"`
6. Push: `git push origin feature/your-feature-name`
7. Open a Pull Request

## Code Style

- Follow ESLint and Prettier configurations
- Use meaningful commit messages
- Write clear, readable code
- Add comments for complex logic

## Reporting Issues

Please use GitHub Issues to report bugs or suggest features.
